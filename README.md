# 异步操作系统训练营学习记录

基于协程异步机制的操作系统/驱动学习。

## 项目背景

本项目用于记录参加异步操作系统训练营的学习过程，包括周报、实验报告、代码改进等内容。

## 项目结构

```
.
├── reports/           # 报告文档
│   ├── 第一次交流纪要.md
│   └── 个人规划.md

```

## 训练营目标

基于 Rust 语言的异步机制 Future 对设备驱动、调度器、系统调用和 IPC 等内核模块进行异步改造。

### 分层级目标

- **基础目标**：完成前两个阶段练习，撰写清晰的分析文档，完整执行整个流程
- **进阶目标**：在理解代码的基础上，对现有模块进行力所能及的改进
- **优秀目标**：做出具有独到见解的重大改进或实现创新性功能


## 学习计划

### 第一阶段：进程、线程和协程的工作原理（第1-2周）

- 学习进程、线程、协程相关文档和视频
- 实现三种爬虫程序（进程/线程/协程）
- 性能对比分析（延时分布、吞吐率、内存开销）

### 第二阶段：用户态线程和协程（第3-4周）

- 动态跟踪分析执行流状态变迁
- 学习 Tokio Future、绿色线程、协程库
- 执行流机制实践与优先级扩展


## 周报

每周学习进展记录在 GitHub 仓库中。
