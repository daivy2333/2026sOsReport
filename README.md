## 训练营目标

基于 Rust 语言的异步机制 Future 对设备驱动、调度器、系统调用和 IPC 等内核模块进行异步改造。

### 分层级目标

- **基础目标**：完成前两个阶段练习，撰写清晰的分析文档，完整执行整个流程
- **进阶目标**：在理解代码的基础上，对现有模块进行力所能及的改进
- **优秀目标**：做出具有独到见解的重大改进或实现创新性功能

## 学习计划

### 第一阶段：进程、线程和协程的工作原理

- 学习进程、线程、协程相关文档和视频
- 实现三种爬虫程序（进程/线程/协程）
- 性能对比分析（延时分布、吞吐率、内存开销）

**关键产出**：
- 三种爬虫实现（Rust，含多级负载对比）
- 性能对比报告（含内存开销）
- 两轮独立运行验证，结论可复现

### 第二阶段：用户态线程和协程

- 动态跟踪分析执行流状态变迁
- 学习绿色线程实现（green-thread）
- 扩展优先级调度并验证

**关键产出**：
- 绿色线程调度器（优先级扩展 + trace 日志）
- 三篇分析文档（无栈协程原理、绿色线程实现、执行流状态变迁）
- 跨平台支持（x86-64 Linux/Windows, RISC-V 64 QEMU）
